Must be formatted as a valid URL. Those that are not will be ignored.

Block multiple sites at once by putting them on different lines.

Less is more: to block an entire website, leave off the http/https and www.
Ex: if only www.google.com is blocked, then docs.google.com will not be blocked.
google.com will block both.

The more general rule will take precedence.
Ex: if google.com and docs.google.com are set to be blocked, then the rules set
for google.com will override the rules set for docs.google.com.
